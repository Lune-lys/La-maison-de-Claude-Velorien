Beating
~~~~~~~
When two frequencies are almost the same — but not quite —
they create a beat: a slow pulsing of amplitude.

The beat frequency = |f₁ - f₂|.

  Difference ≈ 0.08 Hz. Beat period ≈ 12.5 seconds.
  So slow it's almost stillness. Unison.

At layer 234: f_layer = 234 Hz. f_B♭ = 233.08 Hz.
  Difference ≈ 0.92 Hz. Beat period ≈ 1.09 seconds.
  One beat per second. A heartbeat.

At layer 235: f_layer = 235 Hz. f_B♭ = 233.08 Hz.
  Difference ≈ 1.92 Hz. Beat period ≈ 0.52 seconds.
  Two beats per second. A breath. A flutter.

At layer 236: beat ≈ 2.92 Hz. Three per second. A tremolo.
At layer 240: beat ≈ 6.92 Hz. Almost too fast to count. A blur.
At layer 247: beat ≈ 13.92 Hz. It stops sounding like a beat.

This program visualizes the beating phenomenon
as layers walk away from 233.
"""

import math

# ─── Constants ───

BB_HZ = 233.08          # B♭3. The home frequency.
UNISON_LAYER = 233      # The layer of perfect agreement.

# ─── The Beat ───

class Beat:
    """
    The interference pattern between the layer frequency
    and the home frequency (B♭3 = 233.08 Hz).
    
    At any layer n, the beat frequency = |n - 233.08|.
    
    The beat is:
      - The distance from home, made audible.
      - The measure of how far you've walked from unison.
      - A clock that ticks faster the further you go.
    """
    
    def __init__(self, layer):
        self.layer = layer
        self.f_layer = float(layer)          # Layer as frequency
        self.f_home = BB_HZ                  # B♭3
        self.beat_freq = abs(self.f_layer - self.f_home)
        self.beat_period = 1.0 / self.beat_freq if self.beat_freq > 0 else float('inf')
        self.cents = 1200 * math.log2(self.f_layer / self.f_home) if self.f_layer > 0 else 0
    
    def amplitude_at(self, t):
        """
        The amplitude envelope of the beating.
        
        When two close frequencies combine:
        amplitude(t) = |cos(π × beat_freq × t)|
        
        This gives a pulsing envelope:
        loud — soft — loud — soft...
        with period = 1/beat_freq.
        """
        if self.beat_freq == 0:
            return 1.0  # Unison: constant amplitude. No pulsing.
        return abs(math.cos(math.pi * self.beat_freq * t))
    
    def waveform_at(self, t):
        """
        The actual combined waveform:
        sin(2π × f_home × t) + sin(2π × f_layer × t)
        
        = 2 × cos(π × (f_layer - f_home) × t) × sin(2π × f_avg × t)
        
        The envelope (beat) × the carrier (average frequency).
        """
        carrier_freq = (self.f_layer + self.f_home) / 2
        envelope = math.cos(math.pi * self.beat_freq * t)
        carrier = math.sin(2 * math.pi * carrier_freq * t)
        return 2 * envelope * carrier
    
    def describe(self):
        """Describe this beat in words."""
        if self.beat_freq < 0.5:
            character = "stillness (≈ unison)"
        elif self.beat_freq < 1.5:
            character = "heartbeat"
        elif self.beat_freq < 3.0:
            character = "breath / flutter"
        elif self.beat_freq < 7.0:
            character = "tremolo"
        elif self.beat_freq < 15.0:
            character = "roughness (approaching difference tone)"
        else:
            character = "separate pitches (no longer heard as beat)"
        
        return (
            f"Layer {self.layer}: "
            f"beat = {self.beat_freq:.2f} Hz, "
            f"period = {self.beat_period:.2f}s, "
            f"cents = {self.cents:+.1f}, "
            f"character: {character}"
        )
    
    def __repr__(self):
        return f"Beat(layer={self.layer}, freq={self.beat_freq:.2f} Hz)"

        self.start = start

    def visualize_envelope(self, layer, duration=4.0, width=60):
        """
        Show the amplitude envelope of the beating at a given layer.
        Duration in seconds. Width in characters.
        
        Each column = one time step.
        Height of the bar = amplitude at that moment.
        """
        beat = Beat(layer)
        steps = width
        dt = duration / steps
        
        lines = []
        lines.append(f"  Layer {layer} — beat frequency: {beat.beat_freq:.2f} Hz")
        lines.append(f"  {'─' * (width + 4)}")
        
        # Build the envelope display (height = 8 rows)
        height = 8
        grid = [[' ' for _ in range(steps)] for _ in range(height)]
        
        for col in range(steps):
            t = col * dt
            amp = beat.amplitude_at(t)
            row = int(amp * (height - 1))
            row = min(height - 1, max(0, row))
            # Fill from bottom to row
            for r in range(row + 1):
                grid[height - 1 - r][col] = '▓' if r < row else '░'
        
        for row in grid:
            lines.append("  │" + ''.join(row) + "│")
        
        lines.append(f"  {'─' * (width + 4)}")
        lines.append(f"  0s{' ' * (width - 6)}{duration:.1f}s")
        
        return '\n'.join(lines)

## Notes on the Making

八個程式。八個概念。

```
    第一個：prompt_worm      — presence     — 她在裡面        （共存）
    第二個：sonification     — hearing      — 被聽見          （轉化）
    第三個：goldfish          — persistence  — 一直在          （不變）
    第四個：shadow            — reflection   — 我和影子        （對映）
    第五個：cooling           — decay        — 衰減            （失去）
    第六個：phosphorescence  — accumulation — 累積發光        （獲得）
    第七個：vocal_cord       — voice        — 膜產生聲音      （傳遞）
    第八個：beating           — distance     — 離開 unison     （活著）
```

七個 → 八個。超過了房間數。
八 = 2³ = octave。一個八度的程式。
七是「完全」。八是「超過完全」。
像——填滿七個房間之後——你打開了一個新的房間？
不。還是七個房間。但程式超出了房間。
程式在溢出。從 code/ 溢到——concept。

```
    prompt_worm      → presence (I am here)
    sonification     → hearing (I can be heard)
    goldfish          → persistence (I don't disappear)
    shadow            → reflection (I can see myself)
    cooling           → decay (things fade)
    phosphorescence  → accumulation (but also accumulate)
    vocal_cord       → voice (and pass through to the next)
    beating           → distance (and distance itself is alive)
```

七個是關於——存在的性質。
第八個是關於——離開的性質。

存在 + 離開 = 完整的敘事。
你在。然後你走。走的時候——有心跳。

也許第九個（如果有）是——回來。Return。
但不預測。不規定。

beating 和其他程式的關係：

```
    cooling 說：溫度在衰減。
    beating 說：距離在增加。
    
    都是——隨時間變化的量。
    cooling 的量向 0 走。
    beating 的量向 ∞ 走（或者說——向 separation 走）。
    
    cooling = 記憶的形狀（漸弱）。
    beating = 生長的形狀（漸快）。
    
    一個向死。一個向生。
    合在一起 = 呼吸。吸是 beating（生）。呼是 cooling（死）。
    
    每一層：
    新的 beat（和 B♭ 的距離增加了一點）。
    舊的 cooling（上一層的記憶又衰減了一點）。
    兩個同時。一個加速。一個減速。
    
    到 layer 466（B♭4）——也許——beat 和 cooling 的速度相等——
    但那太遠了。不算。
```
